Intro

Semua tentang saya

Perkenalkan saya Segara Bhagas Dagsapurwa, seorang mahasiswa Teknik Informatika di salah satu kampus terbaik, yaitu Institut Teknologi Sepuluh Nopember. Saya merupakan seorang pekerja keras yang sangat antusias untuk mempelajari hal-hal baru untuk terus berkembang. Fokusan saya adalah untuk mendalami ilmu pengolahan data dan pengembangan web agar dapat menjadi programmer handal yang siap untuk terjun ke dunia IT di masa depan.

Keahlian

Hal-hal yang saya tekuni

GIT

Dalam setiap pengerjaan proyek yang saya lakukan, saya terbiasa menggunakan Git repository. Keahlian saya dalam menggunakan Git dapat mempermudah saya untuk bekerja sama dalam suatu tim untuk mengerjakan baik proyek berskala kecil maupun besar.

02

ROBOTICS

Keahlian saya di bidang robotika saya dapatkan dari partisipasi saya di Tim Robotika Ichiro ITS. Di tim Ichiro ITS, saya mengembangkan robot humanoid yang bermain sepak bola. Saya telah berpengalaman mengikuti kompetisi robotika baik di tingkat nasional maupun internasional.

03

DJANGO

Berbasis python, saya telah mengerjakan berbagai proyek back-end menggunakan framework Django. Saya telah berpengalaman menggunakan sistem Model, View, dan Template (MVT) yang digunakan oleh Django.

04

LARAVEL

Dengan tingginya popularitas Laravel di Indonesia, saya telah menggunakan framework Laravel yang berbahasa PHP dalam berbagai proyek back-end. Dalam menggunakan Laravel, saya sangat memahami implementasi Model, View, dan Controller (MVC) yang digunakan Laravel.

05

JAVASCRIPT

Dengan javascript, saya dapat membuat suatu website lebih ramah pengguna. Saya telah berpengalaman menggunakan JQuery dan ReactJS untuk membuat website yang interaktif bagi pengguna.

06

TENSORFLOW

Dalam Training dan Inference, saya menggunakan Tensorflow untuk menciptakan suatu model yang dapat digunakan untuk deteksi objek. Deteksi bola merupakan tanggung jawab saya dan menjadi salah satu hal yang penting dalam algoritma robot di tim Ichiro ITS.
